# EOG claim matrix

| Quantity | Computed quantity | Defensible wording | Required assumptions | Prohibited wording | Evidence status |
|---|---|---|---|---|---|
| Standardized span | Default 0.90 quantile of positive pairwise distances after within-cloud median/MAD scaling | Standardized dispersion of the observed cloud | Predeclared features; finite values; one cloud interpreted in its own scaled coordinates | Absolute niche breadth across independently scaled groups | Implemented; comparative dilation benchmark falsified the absolute-breadth interpretation |
| Comparative span | Span for both groups after one frozen shared-reference transform | Comparative observed extent in declared reference units | Same reference and feature order; valid reference mode; no outcome-driven selection | Universal tolerance, occupied range boundary, suitability breadth | Implemented; external, training-only, and pooled-descriptive modes audited |
| Log span ratio | Median matched-size `log(span_B / span_A)` | Magnitude and direction of comparative extent | Same frozen reference; matched-size draws; positive spans | Population effect without sampling qualification | Implemented; known two-fold dilation recovered in frozen benchmarks |
| `continuity` | Environmental diameter divided by total MST length | MST compactness within a declared support class | Comparable support dimension and sampling design | Generic path tortuosity or connectivity across arbitrary clouds | Implemented; unrestricted path-directness interpretation falsified |
| `gap_strength` | Largest positive MST edge divided by median positive MST edge | Descriptive separation evidence | Declared features; sample-size and support sensitivity reported | Fragmentation probability, missing-support proof, universal cutoff | Implemented; sample-size dependence and missing-bridge weakness verified |
| Component count | Number of graph components after the declared MST-edge threshold | Threshold-dependent diagnostic components | Threshold and scaling reported | Ecological populations, habitat patches, true disconnected niches | Diagnostic only |
| Subsampling interval | Default 0.05 and 0.95 quantiles of matched-size occurrence draws | Sensitivity to the observed occurrence sample | Same draw size in both groups; frozen fraction and seed | Population confidence interval, bias correction, posterior interval | Implemented; matched-size redesign passed null and known-effect audits |
| Direction stability | Fraction of matched-size draws preserving the median effect sign | Resampling stability of the observed direction | Frozen resampling design | Probability the ecological hypothesis is true | Implemented |
| Permutation diagnostic | Two-sided plus-one label-permutation value | Exchangeability-based diagnostic support | Pooled rows exchangeable under the design, or an appropriate restricted scheme | Causal test, posterior probability, valid test under clustered dependence | Implemented with explicit exchangeability boundary |
| Candidate projection | Distance or assignment diagnostic for new feature vectors relative to observed geometry | Diagnostic relation to the observed cloud | Same feature definition and transform | Habitat suitability, occurrence probability, recommended survey site by itself | Core diagnostic only |
| Reference fingerprint | SHA-256 identity of serialized robust-reference parameters | Audit identity for one coordinate system | Deterministic serialization and archived reference payload | Evidence that the reference is ecologically optimal | Implemented |
| Manifest fingerprint | SHA-256 identity of all frozen analytical choices | Audit identity for the predeclared analysis | Manifest frozen before outcomes; matching inputs and reference | Preregistration proof when the manifest was created after inspection | Implemented |

## Historical wording to remove

The following phrases are superseded or unsupported unless used explicitly to describe failed historical hypotheses:

- absolute environmental breadth from independently scaled `span`;
- generic environmental path directness from `continuity`;
- fragmentation detector or missing-bridge detector from `gap_strength`;
- confirmatory fragmentation probability from connected-null calibration;
- universal EOG threshold;
- composite headline EOG score;
- candidate suitability score.
